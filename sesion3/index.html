Memoria

Descripción de la práctica

Esta práctica consiste en la creación de varias escenas 3D utilizando la biblioteca Three.js. Se parte de un cubo básico al que se le aplica una textura avanzada, a diferencia de la práctica anterior. Posteriormente, se desarrolla un ejemplo similar en el que se emplean vídeos como textura en lugar de imágenes estáticas. Por último, se incorporan controles para la interacción con la escena, permitiendo tanto la modificación de parámetros (como el bumpScale) como el control del movimiento en la escena, como en el caso de la última implementación.

Ejercicio 1

En el primer ejercicio se parte del código de un ejercicio de la primera práctica. En él hay un cubo al que se le aplica una textura avanzada, por lo que se utilizan dos imágenes en lugar de una. No se encontró dificultado porque, además de partir de código anterior, en el enunciado se da todo lo necesario para hacerlo sin problemas.

Ejercicio 2

En este segundo ejercicio se introduce el uso de vídeos como texturas. Para ello, se crea un elemento canvas que se actualiza con el contenido del vídeo en cada frame, y se utiliza este canvas como textura para un cubo. La principal dificultad fue entender cómo sincronizar la actualización del canvas con la reproducción del vídeo, pero una vez comprendido el proceso, la implementación fue relativamente sencilla.

Ejercicio 3

Para este ejercicio se construye otra esfera y se le aplica una textura que simula la atmósfera. Lo importante es que el material ha de ser MeshLambertMaterial con transparent: true, para que se vea correctamente. En este ejercicio, tampoco se encontró mucha dificultad.

Ejercicio 4

En este ejercicio se introduce la inclinación del eje terrestre aplicando rotación tanto a la esfera como al grupo que la contiene. La esfera y la atmósfera se agrupan en un en earthGroup, lo que permite aplicar la transformación sin necesidad de repetirla en cada geometría individualmente.
